template<class COMPONENT>
struct gip::RGBAPixel< COMPONENT >

True color pixel with alpha component.

Pixel with red, green, blue, and alpha components. By convention the alpha component specifies the level of opacity with the maximum and minimum value meaning respectively fully opaeque and fully transparent. If the alpha component is given a different meaning it should be explicitly described in the documentation.
